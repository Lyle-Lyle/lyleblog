## Installation & Usage 🚀

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

### 2. Install dependencies


```bash
# Using npm
npm install

# Or using yarn
yarn install
```

### 3. Run the project
```
npm start
```
